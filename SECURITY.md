# Security Policy

If you discover a security issue, please do not open a public issue right away.

## Reporting

Please report vulnerabilities privately to the maintainer first and include:

- A clear description of the issue
- Reproduction steps or a proof of concept
- The expected impact
- Any suggested mitigation if you have one

## Response

Reasonable efforts will be made to confirm the issue, assess impact, and prepare a fix before public disclosure.

## Scope

This project handles:

- Authentication flows
- Session handling
- Database-backed content creation

Please report issues related to auth, session management, database access, and sensitive configuration handling.
